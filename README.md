# 🎓 AI Career Prediction System

An AI-powered web application that helps students discover suitable career paths based on their skills, interests, and aptitude. The platform provides personalized career recommendations through skill assessments and intelligent analysis, helping students make informed career decisions.

## ✨ Features

- 🔐 Secure User Authentication (Register/Login)
- 👤 Student Profile Management
- 🛣️ Manual Career Selection Option
- 🧠 AI-Based Career Prediction
- 📝 Skill Assessment Quiz
- 📊 Personalized Career Recommendations
- 🎯 Skill Gap Analysis
- 🗺️ Career Learning Roadmap
- 💾 Quiz History Storage
- 🤖 AI-Powered Dynamic Question Generation

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/AI-Career-Prediction.git
```

### 2. Move to project directory

```bash
cd AI-Career-Prediction
```

### 3. Create virtual environment

```bash
python -m venv venv
```

### 4. Activate virtual environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux/macOS**

```bash
source venv/bin/activate
```

### 5. Install dependencies

```bash
pip install -r requirements.txt
```

### 6. Configure API Key

Create a `.env` file and add:

```env
ANTHROPIC_API_KEY=your_api_key
```

### 7. Run the application

```bash
python app.py
```

The application will start at:

```
http://127.0.0.1:5001
```
